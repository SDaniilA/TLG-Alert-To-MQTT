# TLG-Alert-To-MQTT

Telegram Alert Monitor - приложение для Windows, которое перехватывает всплывающие уведомления Telegram, 
озвучивает их через TTS (синтезатор речи) и отправляет в MQTT в формате Meshtastic.

Версия: 0.7.62-nightly
Платформа: Windows 10/11

---

## Возможности

- Мониторинг уведомлений: перехват Toast-уведомлений Telegram через Windows API
- Голосовое оповещение (TTS): озвучка сообщений с русским голосом, настройка скорости и громкости
- MQTT отправка: формат Meshtastic (from, to, channel, type: sendtext, payload, hopLimit)
- Фильтрация: белый список отправителей, черный список, ключевые слова
- Приоритетные слова: прерывание речи при срочных сообщениях (ракета/прилет/взрыв)
- Mute (отключение звука): горячая клавиша M - MQTT продолжает работать
- Горячие клавиши: S (статистика), C (конфиг), R (перезагрузка), Q (MQTT on/off), +/- (громкость)
- Удаление уведомлений: режимы immediate (сразу) / on_exit (при выходе) / never
- Логирование: ротация логов по размеру (10 МБ), таймстемпы, разделители

## Быстрый старт

Требования:
- Windows 10/11
- Установленный .NET 6.0 Runtime (скачать с сайта Microsoft)

Установка:
1. Скачай последний релиз из раздела Releases
2. Распакуй в любую папку
3. Настрой config.json

Настройка доступа к уведомлениям:
При первом запуске программа запросит доступ к уведомлениям Windows:
Настройки -> Конфиденциальность -> Уведомления -> Разрешить "Telegram Alert Monitor"

    "MaxPayloadBytes": 100
  },
  "УдалениеУведомлений": {
    "DeleteMode": "on_exit"
  }
}

## Горячие клавиши

M - Вкл/Выкл звук (Mute) - MQTT продолжает работать
+ / - - Увеличение/уменьшение громкости TTS
S - Статистика (всего, озвучено, отправлено MQTT)
C - Показать текущую конфигурацию
R - Перезагрузить config.json без перезапуска
Q - Вкл/Выкл MQTT отправку
Escape - Остановка программы

## Архитектура (в плане)

Telegram Toast -> Перехват -> Фильтрация -> MQTT (Meshtastic)
                              |
                          TTS озвучка
                              |
                          Логирование

Планируемые улучшения:
- Автоматический MQTT reconnect
- TCP отправка на Meshtastic (через WiFi)
- Serial отправка через USB (COM-порт)
- HealthMonitor с цветными статусами интерфейсов

## Настройка Telegram Desktop

Для корректной работы программы необходимо:

1. Открыть Telegram Desktop
2. Перейти в Настройки (Settings)
3. Выбрать пункт "Уведомления и звук" (Notifications and Sounds)
4. Включить опцию "Показывать всплывающие уведомления" (Show popup notifications)
5. Убедиться, что уведомления не отключены в самой Windows:
   - Настройки Windows -> Система -> Уведомления и действия
   - Найти Telegram Desktop в списке и включить уведомления

Без этих настроек программа не сможет перехватывать сообщения.

## Установка MQTT брокера

Для работы MQTT отправки необходим MQTT брокер. Рекомендуется Mosquitto:

### Windows:
1. Скачать Mosquitto с официального сайта
2. Установить как службу Windows
3. Отредактировать файл `mosquitto.conf`:

listener 1883
allow_anonymous true

4. Запустить службу Mosquitto
